# Инструкция для работы с Markdown

## Выделение текста

Чтобы выделить текст курсивом необходимо его обрамить звёздочками (*). Например, *вот так*

Чтобы выделить текст полужирным,его нужно обрамить двумя звёздочками(**). Например, **вот так**
